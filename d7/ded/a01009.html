Detailed Description

template<class ManagedMemory>
class hh::MemoryManager< ManagedMemory, typename std::enable_if_t<!traits::is_managed_memory_v< ManagedMemory > > >

Definition of MemoryManager, for data that do not derive from MemoryData and/ or are not default constructible.

The class defines all methods used by Hedgehog internally and will only throw errors. Should not be used, just here to define the methods.

Attention
The existence of a memory manager into a Task should be checked before usage.
